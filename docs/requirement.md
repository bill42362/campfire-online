# 營火直播（Campfire Online）需求文件

> 一款 **livestream chatroom game** —— 一個可以與觀眾互動的即時「火堆模擬器」。

---

## 一、專案定位

| 項目 | 說明 |
| --- | --- |
| **產品類型** | Livestream chatroom game，火堆模擬器 |
| **開發方式** | 盡可能全部使用 Claude Code AI 進行開發 |
| **運算硬體** | GPU：NVIDIA 2080Ti |

---

## 二、互動行為（觀眾 → 火堆）

- **送禮加柴**：有人送禮，就依照禮物價值加柴火。
- **餘燼白煙**：沒人送木材、柴火燒完後，持續升起一縷白煙。
- **環境模擬**：可模擬多種天光、天氣與季節。
  - 依照 **CST 時區** 模擬白天與黑夜。
  - **四種季節** 分四天輪流。
  - **五種天氣** 分五天輪流。
  - 合計 **20 天為一個完整循環**。
- **畫面資訊疊層**
  - 顯示 **送禮排行榜**。
  - 把不同平台的直播間聊天室，**整合成一個聊天室** 顯示在螢幕上。
- **聊天室機器人**
  - 在各直播間支援類似 `/help`、`/options` 等機器人指令。
  - 大戶可用機器人指令 **複寫（override）未來 24 小時天氣**。

## 四、模擬內容

### 燃燒模擬
- 進行火焰、木材燃燒等模擬。
- 木材因燃燒而 **局部發黑、碳化、燒紅** 的效果。
- 可能擴充 **原子炭、小樹枝、樹葉** 等其他可燃物。

### 天氣對燃燒的影響
- **下雨**：木材底部沾濕。
- **下雪**：木材頂端局部潮濕，燒得比較慢。

### 聲音模擬
- 乾柴烈火燃燒時產生的 **劈啪作響** 也要模擬出來。

### 結構模擬
- 木材堆脆化，造成餘燼掉落。
- 木材堆結構改變而 **倒塌** 等，也要模擬。
