營火直播：
* 一款 livestream chatroom game, 火堆模擬器
* 要僅可能使用 Claude Code AI 進行所有開發
* GPU: 2080Ti 顯卡
* 行為: 
    * 有人送禮就根據禮物價值加加柴火
    * 沒人送木材燒完就持續升起一縷白煙
    * 還可以模擬多種天光，天氣，季節
        * 照著 CST 時區模擬白天黑夜
        * 四種季節分四天輪流
        * 五種天氣分五天輪流，總共 20 天一個循環
    * 在畫面上顯示送禮排行榜，以及把不同平台的直播間聊天室整合成一個聊天室播在螢幕上
    * 在各直播間支援類似 "/help"，"/options" 等聊天室機器人，甚至大戶可以用機器人指令指定複寫 24hr 天氣
* 直播環境:
    * YouTube: RTMP
    * SWAG: WebRTC, puhser.js
* 模擬內容
    * 進行火焰，木材燃燒等模擬
    * 木材因為燃燒而局部發黑，碳化，燒紅的效果
    * 可能還會擴充原子炭，小樹枝，樹葉等其他可燃物
    * 下雨會讓木材底部沾濕，下雪會讓木材頂端局部潮濕而燒得比較慢
    * 乾柴烈火燃燒時產生的劈啪作響也要模擬出來
    * 木材堆脆化，造成餘燼掉落，木材堆結構改變而倒塌等，也要模擬
